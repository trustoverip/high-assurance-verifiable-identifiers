# High Assurance Verifiable Identifiers (HAVID) Specification

A ToIP specification for High Assurance Verifiable Identifiers.

To browse the spec, see the [rendering on GitHub Pages](https://trustoverip.github.io/high-assurance-verifiable-identifiers/).

## Viewing the spec locally
```
git clone https://github.com/trustoverip/high-assurance-verifiable-identifiers.git
cd high-assurance-verifiable-identifiers
npm install
npm run render
open docs/index.html
```

## Details

The spec is written using Trust Over IP's [Spec-Up-T Tool](https://github.com/trustoverip/spec-up-t).

To contribute to the spec, submit PRs that modify the `.md` files in the `spec/` folder.
